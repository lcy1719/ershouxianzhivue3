### 二手闲置交易系统+Vue3++SpringBoot
## 项目采用技术：
开发环境：IDEA、JDK1.8、Maven、MySQL、VSCode
技术栈：Java、MySQL、SpringBoot、Mybatis/MybatisPlus、Vue3、ElementUI、JavaScript、CSS

系统功能模块：
本系统包含管理员和用户2个角色。
管理员角色：
1. 用户管理：管理系统中所有用户的信息，包括添加、删除和修改用户。
2. 配置管理：管理系统配置参数，如轮播图图片路径等。
3. 商品信息管理：管理商品类型和商品信息，包括新增、查看、修改、删除商品信息。
4. 订单管理：管理商品订单，包括查看、发货、总价统计等操作。
5. 客服聊天管理：管理客服聊天记录，包括查看、回复用户提问。
6. 轮播图管理：管理轮播图配置。
用户角色：
1. 商品信息查看：用户可以查看商品信息及其评论。
2. 购物车管理：用户可以管理购物车中的商品，包括查看、修改、删除购物车中的商品。
3. 客服聊天：用户可以发起客服聊天，向管理员提问。
4. 地址管理：用户可以管理自己的收货地址，包括新增、查看、修改、删除地址。
5. 订单管理：用户可以查看和管理自己的订单，包括已支付、已发货、已取消等订单状态。
6. 我的收藏管理：用户可以管理自己的收藏商品。
